# react
This is a two resource Angular CRUD client app backed by Mongo for data persistence.

## Running

### Install packages

```bash
> npm install
```

### Install gulp

```bash
> npm install --save-dev gulp
```

### Build assets

```bash
> ./node_modules/.bin/gulp
```

### Start mongodb

```bash
> mongod --dbpath=./db
```

### Run server

```bash
> node ./server.js
```

### CHECK IT OUT

```
open http://localhost:5000
```
