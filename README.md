# react
This is a two resource CRUD client app using the Angular framework and backed by Mongo for data persistence.

## Running

### Install packages

```bash
> npm install
```

### Build assets

```bash
> ./node_modules/.bin/gulp
```

### Start mongodb

```bash
> mongod --dbpath=./db
```

### Run backend server

```bash
> node ./server.js
```

### Run client server

```bash
> node ./server.js
```
### CHECK IT OUT

```
open http://localhost:5000
```
